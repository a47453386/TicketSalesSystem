# 售票系統

一套針對高供業務流程設計的發票管理系統，
支援發票開立、QR Code 驗證與自動配位功能。

## 🎬 功能展示影片
👉 [https://canva.link/1odpz69pbqo58w6]

## 🛠 使用技術
- 後端：C# / ASP.NET Core 8
- 資料庫：SQL Server / Entity Framework Core
- 前端：HTML / CSS / JavaScript
- 版本控制：Git / GitHub

## ✨ 主要功能
- 發票開立與管理
- QR Code 自動驗證機制
- 自動配位演算法
- 資料查詢與報表瀏覽

## 💻 本機執行方式
1. Clone 此專案
2. 修改 appsettings.json 中的資料庫連線字串
3. 執行資料庫 Migration
4. 啟動專案
